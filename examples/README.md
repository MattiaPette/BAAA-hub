# Activity Tracker Examples

This directory contains minimal end-to-end and workflow code samples to help you
get started with the Activity Tracker application.

## Available Examples

### 1. [Basic Initialization](./01-basic-initialization.md)

Learn how to set up the basic application structure with all required providers.

### 2. [Authentication](./02-authentication.md)

Understand how to integrate Auth0 authentication, handle login/logout, and
access user information.

### 3. [Theme and Styling](./03-theme-and-styling.md)

Configure theme modes (light/dark) and customize the Material-UI theme.

### 4. [Data Fetching](./04-data-fetching.md)

Make HTTP requests using the `makeRequest` helper and TanStack Query.

### 5. [Navigation and Routing](./05-navigation-and-routing.md)

Work with breadcrumbs, page titles, and navigation blocking.

## Quick Start

If you're new to the project, we recommend following the examples in order:

1. Start with **Basic Initialization** to understand the app structure
2. Move to **Authentication** to learn about user management
3. Explore **Theme and Styling** for UI customization
4. Learn **Data Fetching** for API integration
5. Finally, check **Navigation and Routing** for advanced navigation features

## Prerequisites

Before using these examples, make sure you have:

- Node.js v25.1.0 (or Node 22.x LTS)
- Yarn package manager (v1.22+)
- Basic knowledge of React and TypeScript

For full installation instructions, see the [main README](../README.md).
